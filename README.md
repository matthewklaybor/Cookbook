## About The Project

The Cookbook App for iOS displays a list of meals along with their recipe grouped by food category.

## Built With
Xcode 15, iOS 17.2, Swift, SwiftUI, CoreData

## Getting Started

### Installation
1. Install Xcode 15
2. Clone the repo
3. Run Cookbook App in Xcode
